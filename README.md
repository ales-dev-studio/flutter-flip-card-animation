# flutter_flip_card_animation

This project is a Flutter-based application that demonstrates a flip card animation using custom widgets and animations.
The app allows users to interact with a series of cards, each of which can be flipped to reveal additional information on the back.
The project is designed to showcase how to create visually appealing and interactive UI components in Flutter, such as flip animations, responsive layouts, and dynamic content rendering.
   
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ales-dev-studio/flutter-flip-card-animation.git
   ```

2. Change to the project directory:

   ```bash
   cd flutter-flip-card-animation
   ```

3. Install the dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app:

   ```bash
   flutter run
   ```
